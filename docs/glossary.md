# Glosario de Términos
- **Capa Raw (Bronze):** Datos crudos, exactamente como se extraen de la fuente original. Garantizan la trazabilidad ante cualquier error futuro.
- **Capa Silver:** Datos limpios, normalizados y con tipos de datos correctos. Es la fuente de verdad para el cruce de tablas.
- **Capa Gold:** Datos agregados, filtrados y modelados. Están listos para el análisis estadístico o el cálculo de métricas finales.
- **DuckDB:** Sistema de gestión de bases de datos relacionales en memoria, optimizado para análisis de datos pesados.
